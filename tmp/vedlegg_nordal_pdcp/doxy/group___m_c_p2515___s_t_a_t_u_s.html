ICOD.

Interrupt Flag Code bits <2:0> 000 = No interrupt 001 = Error interrupt 010 = Wake-up interrupt 011 = TXB0 Interrupt 100 = TXB1 Interrupt 101 = TXB2 Interrupt 110 = RXB0 Interrupt 111 = RXB1 Interrupt

Attention:
Read-only!
#define CANSTAT_OPMOD_MASK   ((uint8_t)( (1<<7)|(1<<6)|(1<<5) ))

OPMOD.

Operation Mode bits <2:0> 000 = Device is in the Normal operation mode 001 = Device is in Sleep mode 010 = Device is in Loopback mode 011 = Device is in Listen-only mode 100 = Device is in Configuration mode

Attention:
Read-only!
